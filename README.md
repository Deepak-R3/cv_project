# 🐾 Animal Detection using YOLOv8 and OpenCV

A real-time **Animal Detection System** built using **YOLOv8**, **OpenCV**, and **Python**.  
This project detects multiple animals through a webcam and displays:

- animal name
- confidence score
- bounding box

---

## 📌 Features

✅ Real-time animal detection  
✅ Bounding box display  
✅ Confidence score  
✅ Webcam support  
✅ Easy custom dataset training  
✅ Supports multiple animals  

Detected animals:
- Dog
- Cat
- Cow
- Horse
- Sheep
- Mouse
- More can be added

---

## 🛠 Technologies Used

- Python 3.11
- OpenCV
- YOLOv8
- Ultralytics
- NumPy
